Monitor your website's uptime and performance directly from your WordPress dashboard with PagePatrol.

== Description ==

PagePatrol brings professional website monitoring to your WordPress dashboard. Keep track of your site's uptime, performance, and incidents all in one place.

**Key Features:**

* Real-time uptime monitoring
* Performance tracking
* Incident reporting
* WordPress admin bar integration
* Simple setup process

**Pro Features at PagePatrol.net:**
* Advanced incident analysis
* Custom status pages
* Team collaboration
* Multiple site monitoring
* Advanced notifications

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/pagepatrol` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the PagePatrol menu item to configure the plugin
4. Enter your PagePatrol API key from pagepatrol.net

== Frequently Asked Questions ==

= Do I need a PagePatrol account? =

Yes, you need a PagePatrol account to use this plugin. You can sign up at [pagepatrol.net](https://pagepatrol.net).

= How often is my site checked? =

By default, we check your site every 5 minutes. You can adjust this frequency in your PagePatrol dashboard.

= Is this plugin free? =

The plugin itself is free, but it requires a PagePatrol account to function. Visit our website for pricing details.
